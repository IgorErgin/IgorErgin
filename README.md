# Привет, я Игорь! 👋

### 👨‍💻 Android-разработчик

Я Android-разработчик с упором на **Modern Android Development**. В данный момент руковожу командой разработки в проекте Т-Банка [SmartBudget](https://github.com/TheFourPixels)

Специализируюсь на создании чистых и масштабируемых приложений, используя **Kotlin** и **Jetpack Compose**, уделяя особое внимание архитектуре и качеству кода.
